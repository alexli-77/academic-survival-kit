# Academic Rookie Survival Kit

A small, growing reference site for new PhD students: conference deadline trackers and official conference sites, organized so you stop missing submission windows.

**Live site:** published via GitHub Pages.

## What's here

- **Conference Deadline Trackers** — sites that aggregate submission deadlines across many venues (SE Deadlines, ML Deadlines, AI Deadlines, CSConf, WikiCFP, PLSE Deadlines, AI/ML Conference Tracker, Bibby Conference Deadlines).
- **Conference Sites & CMS** — where individual conferences actually live once you know which one you're targeting (conf.researchr.org, DBLP, OpenReview).

More categories (writing tools, funding/travel grants, self-media & outreach) are planned as this grows.

## Local development

This is a single static `index.html` file with inline CSS/JS — no build step. Just open it in a browser, or serve it locally:

```bash
python3 -m http.server 8000
```
